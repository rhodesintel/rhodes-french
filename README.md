# Rhodes French - Interactive Web App

A complete, free French language course with 7,799 drill exercises and native speaker audio.

## Try It Now

Visit: **https://rhodesintel.github.io/rhodes-french/**

## Features

- 24 units covering practical French conversation
- 7,799 drill sentences with spaced repetition
- Native French audio recordings
- Grammar hints and error analysis
- Progress tracking (saved locally)
- Works offline after first load
